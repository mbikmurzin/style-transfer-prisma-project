# Custom Style Images

Place personal `.jpg`, `.jpeg` or `.png` style references directly in this
folder, then run:

```bash
bash ../../../scripts/04_create_tfrecords.sh
```

For the fast assignment pipeline, DTD is used by default because it is
explicitly referenced in Magenta's arbitrary image stylization instructions.
For a larger experiment, use a licensed subset of WikiArt or Painter by Number
and document its provenance in the report.
