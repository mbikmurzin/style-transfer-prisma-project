# Demo Video Script

Продолжительность: 45-60 секунд.

1. Покажите главный экран приложения и название `Neural Canvas`.
2. Нажмите `Галерея` либо `Камера` для content image и покажите preview.
3. Для style image выберите файл из галереи или один из встроенных примеров.
4. Покажите `Switch GPU`, выбор CPU threads и нажмите `Применить стиль`.
5. После результата покажите блок метрик и сохраните картинку в галерее.
6. В конце покажите `style_predict.tflite` и `style_transform.tflite` в Android assets и один кадр с последовательностью команд из `training/README_training.md`.

Запишите экран телефона или эмулятора в MP4 и положите ссылку на видео в README перед сдачей.
