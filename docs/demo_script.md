# Сценарий демонстрации: 1-2 минуты

Цель видео - показать не макет интерфейса, а фактический inference и материалы
проекта.

| Время | Действие | Что должно быть видно |
| --- | --- | --- |
| 0:00-0:10 | Открыть приложение `Neural Canvas`. | Главный экран, кнопки выбора и настройки inference. |
| 0:10-0:25 | Выбрать фото из галереи или снять камерой. | Content image появляется в первом блоке. |
| 0:25-0:35 | Выбрать стиль из assets или галереи. | Style image появляется во втором блоке. |
| 0:35-0:45 | При желании переключить GPU/CPU threads и нажать **Применить стиль**. | Видна кнопка запуска и ожидание результата. |
| 0:45-1:00 | Показать результат. | В третьем блоке отображается stylized image. |
| 1:00-1:10 | Показать метрики и сохранить результат. | Времена этапов, GPU status, threads и Toast сохранения. |
| 1:10-1:30 | Открыть GitHub README. | Структура проекта, training pipeline, Android/Flask запуск и evidence. |

## Вариант при проблемах с Android

Если APK не удается запустить на устройстве, откройте
`demo/flask-prototype/`, загрузите те же content/style images, выполните
inference и покажите result/metrics в браузере. В README и в речи явно
укажите, что это fallback-прототип, а Android-код реализован отдельно.

## Перед записью

- Убедитесь, что `.tflite` модели установлены.
- Выполните один тестовый inference до начала записи.
- Очистите старые результаты, чтобы в видео было видно новый output.
- Подготовьте открытый `README.md` для финального кадра.
