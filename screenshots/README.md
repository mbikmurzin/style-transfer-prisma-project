# Screenshots

`00_ui_mockup_placeholder.png` уже включен как иллюстративный макет интерфейса. Он
не является доказательством запуска APK и должен быть дополнен реальными кадрами.

После запуска приложения сохраните сюда:

| Filename | Required frame |
| --- | --- |
| `01_home_screen.png` | Экран приложения до выбора изображений. |
| `02_inputs_selected.png` | Загруженные content и style previews. |
| `03_stylized_result.png` | Результат после TFLite inference. |
| `04_saved_gallery.png` | Файл результата в галерее устройства. |

Для отчета достаточно первых трех кадров; четвертый подтверждает экспорт результата пользователю.
